## **使用github與jekyll建立自己的blog**


# **0.** Requirements check

    ruby -v  
    gem -v  
    gcc -v  
    g++ -v  
    make -v  

# **1.** Jekyll

    gem install jekyll
    jekyll new [dir_name]
    cd [dir_name]
    
# **2.** Github
    
    git init
    git add .
    git commit -m "git init"
    git remote add origin [github url]
    git push
